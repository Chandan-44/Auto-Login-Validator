## 🧠 Overview

**Auto Login Validator** is a browser-based tool designed to automate **login form testing using username and password lists**.

It helps security testers and developers validate authentication mechanisms and detect:

* Weak credentials
* Improper login validation
* Authentication logic flaws

> ⚡ Designed for controlled environments and authorized testing only

---

## 🚀 Key Features

### 📄 Smart Form Detection

* Automatically detects login forms on a webpage
* Allows manual selection of target form

---

### 🎯 Flexible Input Selectors

* Username field selector
* Password field selector
* Submit button selector

📌 Works on almost any login UI

---

### 📂 Credential List Support

* Upload username list (.txt)
* Upload password list (.txt)
* Automatic iteration (user × password)

---

### ⚡ Automated Validation Engine

* Attempts login combinations automatically
* Tracks progress in real-time

---

### 🧠 Smart Success Detection

Detects successful login using:

* URL pattern match (e.g. `/dashboard`)
* DOM element detection (e.g. `.welcome-msg`)

---

### ❌ Failure Detection

* Detects error messages (e.g. `.invalid-login`)
* Stops false positives

---

### 📊 Real-Time Logs

* Success logs
* Failure logs
* Progress tracking

---

## 🧪 Example Workflow

1. Open target website
2. Launch extension
3. Select login form
4. Upload:

   * Username list
   * Password list
5. Configure:

   * Success condition
   * Failure condition
6. Click **Start Validation**
